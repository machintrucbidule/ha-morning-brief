# Morning Brief

AI-narrated daily / evening / weekly briefs for Home Assistant, built from
your own sensors and calendars.

- 8 field provider types (energy, instantaneous, events, state, duration, calendar, weather, manual)
- 8 comparison types per field, 3 anomaly modes
- 3 logical-day strategies (fixed cutoff, sleep sensor, manual)
- 4 AI provider backends (HA ai_task, Anthropic direct, OpenAI direct, disabled)
- FR + EN at launch
- Companion Lovelace card: [`ha-morning-brief-card`](https://github.com/machintrucbidule/ha-morning-brief-card)

See the README for setup, screenshots, and the architecture docs.
